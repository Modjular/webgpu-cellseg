Cell segmentation in the browser, on WebGPU

Three microscopy segmentation models ported to hand-written WGSL — no PyTorch, no TensorFlow, no CUDA, no ML framework at inference time. Serve this folder over http://localhost and open a demo (WebGPU required — Chrome/Edge, or Safari 18+).

Cellpose cyto3

Generalist cytoplasm segmentation. CNN forward + flow dynamics, with cellpose-exact tiled inference — matches desktop Cellpose at AP@0.5 = 1.0.

StarDist 2D_versatile_fluo

Star-convex nuclei segmentation. U-Net → radial distances → polygon NMS. Best on fluorescence-nuclei images.

InstanSeg brightfield_nuclei

Embedding-based instance segmentation. U-Net → learned pixel embeddings → instance decode.

Source, architecture notes, and the hard-won gotchas: see the repo README and docs/GOTCHAS.md. Weights load from ../weights/<model>/ by default; point Model.load(url) at a CDN to serve them elsewhere.
